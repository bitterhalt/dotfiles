# Dotfiles
This is where I keep all of my dotfiles for apps that I use daily. Feel free to copy any of the dotfiles or scripts you find here. 🙃

### To replicate my Arch setup

1. clone repo
2. install all needed packages: ```pacman -S --needed $(comm -12 <(pacman -Slq | sort) <(sort pkglist))```
3. ```chmod +x``` all the scripts from **.local/bin** and **~/.config/polybar/scripts/**
